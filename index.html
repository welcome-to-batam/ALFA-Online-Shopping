Install Aplikasi Website ALFA Online Shopping

cara install di hp melalui Browser Chrome: Buka ALFA Online Shopping di Browser Chrome. Klik titik tiga di pojok kanan atas. Klik "Tambahkan Ke-layar Utama. Klik Install. Klik Install sekali lagi, lalu tunggu satu menit. Setelah itu akan muncul pemberitahuan di atas layar hp, "ALFA Online Shopping Sekarang Aplikasi Diinstall". Tutup browser chrome dan kembali ke layar beranda utama Anda, cari aplikasi web bernama ALFA. Setelah itu, mainkan dompet Anda.
